# MHNs

This directory contains trained MHN MLEs.
The model and their metadata are named as follows:
```
<dataset_name>_<prior_name>_mle.csv
```
and
```
<model_name>_meta.json
```
They were trained as described in the [manuscript](../../docs/Quantifying_Uncertainty_of_Predictions_from_Cancer_Progression_Models.pdf).
